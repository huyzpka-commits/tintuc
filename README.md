# VietNews — Tin tức tổng hợp

Trang tin tức tổng hợp đơn giản, hiện đại, responsive với giao diện sáng/tối.

## Demo

Website đã được publish qua GitHub Pages:

👉 **[https://huyzpka-commits.github.io/tintuc/](https://huyzpka-commits.github.io/tintuc/)**

## Công nghệ

- HTML5
- Tailwind CSS (qua CDN)
- JavaScript thuần

```
.
```

## Mô tả

VietNews là một trang tin tức tổng hợp với thiết kế tối giản, hỗ trợ dark mode, tìm kiếm và hiển thị tin tức theo danh mục. Dự án chạy hoàn toàn trên frontend, không cần backend.

## Triển khai

Dự án được tự động triển khai qua **GitHub Pages** từ branch `main`.

## Giấy phép

Dự án nguồn mở cho mục đích học tập và tham khảo.
